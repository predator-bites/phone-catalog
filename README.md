
# Gadgets shop 

Shop site for phones, tablets, watches, accessories.
## ✨ Introduction
Educational project to sort out knowledge of TSX (Typescript on React), routing, saving history of shopping in browser. Project brougt to me expirience of creating multipage app with adaptive design and huge amount of details that I cant mention in readme.

## 💦  Features

- Adaptive design
- Routing between 7 pages
- Userfriendly design
- Custom pagination
- Memorized busket and favorited products
- Fast loading with lazy imports
- 7 sorting variants
## 💻 Technologies

#### Core

- React 18 — UI library
- TypeScript 5 — static typing
- Vite 5 — build tool & dev server

#### Routing & State

- React Router DOM v6 — client-side routing
- React Context API — global state (cart, favorites, sidebar, etc.)

#### Styling

- SCSS / Sass — CSS preprocessor with modules
- Lucide React — SVG icon components

#### UI / Utilities

- classnames — conditional class binding
- lodash — utility functions
- react-swipeable — touch/swipe gestures (slider)

#### Tooling & Quality

- ESLint (Airbnb TypeScript config) — linting
- Prettier — code formatting
- Stylelint — SCSS linting
- Path aliases — better code readability


## 🌐 Installation

Installs project from github and installs necessary dependencies

```bash
    git clone https://github.com/predator-bites/gadgets-catalogue.git && npm install  
```

Run project locally

```bash
    npm start
```
